隐私政策

最后更新:2026 年 7 月 6 日

EasyTrail 致力于保护用户隐私。我们坚持最小化数据收集原则,本工具不收集、不存储、不传输任何与您个人健康或身份相关的信息。

一、我们不收集的信息

本工具不收集以下任何信息:

  • ❌ 您的姓名、邮箱、电话等个人身份信息
  • ❌ 您的搜索历史、浏览记录
  • ❌ 您的健康状况、疾病信息
  • ❌ 您的 IP 地址(服务器端不记录)
  • ❌ 您的位置信息
  • ❌ 您的设备指纹
  • ❌ 您的任何行为数据

二、我们使用的本地存储

为了提供更好的浏览体验,本工具在您的浏览器本地(使用 localStorage)保存以下信息:

  • 您的界面偏好(语言、显示模式等)
  • 您收藏的试验清单(可选,您可随时清除)

这些数据仅存储在您的浏览器中,不会上传到我们的服务器。您可随时通过浏览器设置清除。

三、第三方服务

本工具不使用任何第三方追踪、分析或广告服务。具体来说:

  • ❌ 不使用 Google Analytics / 百度统计 / 其他分析工具
  • ❌ 不使用任何广告网络
  • ❌ 不嵌入任何第三方追踪脚本
  • ❌ 不使用第三方 Cookie

本工具的 API 请求直接发送到我们的 Cloudflare Workers,不经过任何第三方代理。

四、数据来源

本工具的所有试验数据来源于 ClinicalTrials.gov(美国国立卫生研究院公开数据库)。这些数据是公开数据,不包含任何用户隐私信息。

当您使用本工具时,您的浏览器会向我们的 Cloudflare Workers 发送搜索请求(仅包含您输入的关键词和筛选条件),Workers 返回试验数据。这一过程中,我们不记录您的 IP、请求时间戳或任何识别信息。

五、Cookies

本工具不使用任何 Cookies 进行用户追踪。本工具仅可能使用浏览器本地存储(localStorage)保存您的界面偏好,这与第三方追踪无关。

六、儿童隐私

本工具面向成年用户。我们不会故意收集任何 14 岁以下儿童的信息。如果您是儿童监护人,发现孩子在未经您同意的情况下使用了本工具,请理解本工具不收集任何儿童信息。

七、数据安全

虽然我们不收集用户数据,但我们仍然采取了行业标准的安全措施保护本工具本身:

  • 使用 HTTPS 加密所有通信
  • Cloudflare Workers 提供 DDoS 防护
  • 源代码公开,接受社区监督

八、政策的变更

我们可能根据需要更新本隐私政策。更新后的政策将在本页面发布,变更会显著标注。继续使用本工具即视为接受新政策。

九、您的权利

由于我们不收集您的任何个人信息,您无需也无法行使通常意义上的"删除数据"权利。

如果您对本工具有任何隐私方面的疑虑,可通过项目 GitHub Issues 提交反馈。

十、联系我们

如对本隐私政策有任何疑问,可通过项目 GitHub Issues 提交反馈。

用户协议

最后更新:2026 年 7 月 6 日

请仔细阅读本协议。使用 EasyTrail(以下简称"本工具")即表示您同意本协议全部条款。如果您不同意任何条款,请立即停止使用本工具。

一、工具性质声明

本工具是一个临床试验信息检索工具,所有数据来源于 ClinicalTrials.gov(美国国立卫生研究院维护的公开临床试验数据库)及其他公开来源。

本工具仅提供信息查询功能,不对任何疾病作出诊断、治疗建议或医疗指导。本工具所列临床试验的参与资格、入组标准、风险与收益等具体信息,均应以申办方及临床试验机构官方说明为准。

二、医疗免责声明

使用本工具查阅临床试验,不构成医患关系,不替代专业医疗咨询。任何参与临床试验的决定,均应在充分咨询主治医生、独立了解试验详情、并签署知情同意书后作出。

本工具开发者不对以下情况承担责任:

  • 临床试验信息变更、撤销或终止
  • 用户因使用本工具作出任何医疗决策导致的任何后果
  • 因数据源延迟、错误或缺失导致的任何问题
  • 任何第三方因使用本工具信息而产生的损失

三、用户行为规范

您承诺不会将本工具用于以下用途:

  • 任何违法违规的活动
  • 对他人作出医疗建议或诊疗指导(本工具仅供查询)
  • 商业转售本工具数据(数据本身为公开数据,但批量转售需联系开发者授权)
  • 干扰本工具正常运行或破坏其完整性
  • 利用本工具从事任何医疗执业活动

四、数据来源与准确性

本工具数据来源于 ClinicalTrials.gov(美国国立卫生研究院公开数据库)。数据每 6-12 小时自动同步更新。

虽然我们努力确保数据准确性,但不保证所有信息均完整、准确、及时。试验状态、入组标准、招募人数等可能随时变更。请始终以 ClinicalTrials.gov 官方页面及试验申办方信息为准。

五、知识产权

本工具的源代码遵循 MIT 开源协议(查看许可证)。临床试验数据的所有权归 ClinicalTrials.gov 及其原始申办方所有。

本工具为信息聚合工具,不主张对底层试验数据的所有权。如您是试验申办方并希望从本工具中移除特定试验数据,请通过项目 GitHub Issues 联系开发者。

六、隐私保护

本工具不收集任何与个人健康相关的隐私信息。详见 隐私政策。

七、服务变更与终止

开发者保留随时修改、暂停或终止本工具的权利,无需提前通知。

八、协议变更

开发者可能根据需要更新本协议。更新后的协议将在本页面发布,继续使用本工具即视为接受新协议。

九、法律适用

本协议的解释、效力及争议解决均适用中华人民共和国法律。

十、联系我们

如对本协议有任何疑问,可通过项目 GitHub Issues 提交反馈。
